# Afik Yefet - AI Crypto Advisor

AI Crypto Advisor is a React and Node.js web app that serves as a personalized crypto investor dashboard.
The app gets to know the user through a short onboarding quiz and then shows
daily AI-curated content tailored to their interests.

## Installation and getting started

Use the package manager npm to install dependencies.

### frontend
```bash
cd "frontend"
npm install
```
### backend
```bash
cd "backend"
npm install
```
## running the web app

### frontend
```bash
cd "frontend"
npm run dev
```
### backend
```bash
cd "backend"
npm run dev
```
